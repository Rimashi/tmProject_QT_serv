Detailed Description

Окно администратора для отображения статистики тестирования.

Класс окна администратора для отображения статистики тестирования.

Представляет собой виджет с двумя панелями:

  • Левая панель с меню (пользователи, статистика, выход).
  • Правая панель с таблицей результатов тестирования студентов.

Этот класс представляет графический интерфейс, в котором администратор может просматривать результаты прохождения тестов пользователями. Интерфейс включает таблицу со статистикой и навигационное меню.
