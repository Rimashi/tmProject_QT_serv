\doxysubsection{Detailed Description}
Модель данных \doxylink{class_reg_window}{Reg\+Window} (шаблон на основе QAbstract\+Item\+Model). 

Класс модели данных \doxylink{class_reg_window}{Reg\+Window}.

Эта модель предназначена для работы с пользовательскими данными, но пока не реализована. Может быть расширена для отображения в QTree\+View/\+QTable\+View.

Этот класс наследует QAbstract\+Item\+Model и представляет собой шаблон модели, который можно использовать в представлениях (например, QTree\+View или QTable\+View). Пока не реализует конкретную структуру данных.
