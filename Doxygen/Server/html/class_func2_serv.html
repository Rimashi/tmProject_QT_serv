QString Func2Serv::extractBitsFromImage ( const QImage & image,
int bitCount )

Извлекает битовую строку из изображения методом LSB.

Из каждого пикселя берёт младший бит каждого цветового канала (R, G, B) по порядку, пока не будет извлечено указанное количество бит.

Parameters
imageИзображение формата QImage, из которого извлекаются данные.
bitCountКоличество бит, которое нужно извлечь.
Returns
Извлечённая строка, содержащая символы '0' и '1'.
