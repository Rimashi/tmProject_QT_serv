Detailed Description

Реализация алгоритма шифрования AES-128 (Advanced Encryption Standard)

Класс предоставляет статические методы для генерации ключей, шифрования и дешифрования данных с использованием 128-битного ключа и режима ECB (Electronic Codebook).

Warning
Не используйте ECB режим для чувствительных данных! Для реальных проектов рекомендуется реализовать CBC или другие более безопасные режимы.

Алгоритм:

  1. Проверка корректности размеров
  2. Генерация раундовых ключей
  3. Обработка блоков по 16 байт (в обратном порядке):
    • Начальный AddRoundKey (с последним ключом)
    • 9 основных раундов (InvShiftRows, InvSubBytes, AddRoundKey, InvMixColumns)
    • Финальный раунд (без InvMixColumns)
  4. Удаление PKCS#7 padding
