\doxysubsection{Detailed Description}
Реализация алгоритма шифрования AES-\/128 (Advanced Encryption Standard) 

Класс предоставляет статические методы для генерации ключей, шифрования и дешифрования данных с использованием 128-\/битного ключа и режима ECB (Electronic Codebook). \begin{DoxyWarning}{Warning}
Не используйте ECB режим для чувствительных данных! Для реальных проектов рекомендуется реализовать CBC или другие более безопасные режимы. 
\end{DoxyWarning}
