\doxysubsection{Detailed Description}
Класс TCP-\/сервера для обработки клиентских подключений и работы с БД. 

Обрабатывает входящие подключения, обмен ключами шифрования (\doxylink{class_r_s_a}{RSA} + \doxylink{class_a_e_s}{AES}), а также маршрутизирует действия клиента через \doxylink{class_func2_serv}{Func2\+Serv}.
